# End to End Cricket Data Analytics Project

An end-to-end data analytics project focused on cricket match data. This project collects raw cricket data, processes it using Python, performs data analysis, and builds interactive visualizations and dashboards.

The goal of this project is to demonstrate a complete data analytics pipeline including data ingestion, preprocessing, analysis, visualization, and deployment using DevOps practices.
